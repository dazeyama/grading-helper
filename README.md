# grading-helper

A static, client-side JavaScript web app hosted on GitHub Pages.

- **Live:** https://dazeyama.github.io/grading-helper/
- **Stack:** pure client-side — `index.html` + `styles.css` + `app.js` + `config.js`. No backend (must stay static for GitHub Pages).

## Run locally

```
python -m http.server 8090 --directory C:\ClaudeProjects\grading-helper
```

Then open http://localhost:8090.
